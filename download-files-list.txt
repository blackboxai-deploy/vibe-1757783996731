## Cara Download:

### Opsi 1: Manual Copy-Paste
1. Kunjungi website live: https://sb-2rkbgt3osn33.vercel.run
2. Gunakan browser developer tools untuk melihat source files
3. Copy setiap file sesuai struktur di atas
4. Gunakan file konfigurasi yang telah dibuat (package-download.json, dll)

### Opsi 2: Clone & Setup
1. Jalankan setup-project.sh untuk struktur dasar
2. Copy semua source files sesuai daftar di atas  
3. Install dependencies dengan npm install
4. Test dengan npm run dev

### Opsi 3: Create Next.js + Manual Files
1. npx create-next-app@latest zathaya-soft --typescript --tailwind --eslint --app --src-dir
2. Copy semua file sesuai struktur
3. Install additional dependencies dari package-download.json
4. Setup shadcn/ui components

## File Sizes (Perkiraan):
- Total project size: ~50MB (termasuk node_modules)
- Source files only: ~500KB
- Main pages: ~200KB
- Components: ~150KB
- Config files: ~50KB

## Prioritas Download:
1. ⭐ WAJIB: package-download.json, README-DOWNLOAD.md
2. 🔧 PENTING: All src/ files (layout.tsx, page.tsx, components, etc.)
3. ⚙️ KONFIGURASI: next.config.ts, tailwind.config.ts, tsconfig.json
4. 📚 DOKUMENTASI: DRUPAL-INTEGRATION.md, setup-project.sh
5. 🎨 ASSETS: public/ files (opsional, bisa diganti)

## Setelah Download:
1. Run setup-project.sh atau manual setup
2. Copy semua source files ke struktur yang sesuai
3. npm install untuk install dependencies  
4. npm run dev untuk test development
5. npm run build untuk production build
